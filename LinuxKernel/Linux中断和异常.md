# Linux中断

## 为什么会有中断

+ 内核的主要功能就是处理外设的IO
  + 处理器的速度一般比外设快很多
  + 加入中断后可以处理其他任务，完成后CPU才处理外设IO
  + 轮询方式效率低

+ 查看系统中断信息
  + cat /proc/interrupts

## 中断和异常

中断（广义）会改变处理器执行指令的顺序，通常与CPU芯片内部或外部硬件电路产生的电信号相对应，用"中断信号"来统称这两类类型的中断。

+ 中断----异步
  + 由硬件随机产生
+ 异常----同步
  + 在（特殊的或出错的）指令执行时由CPU控制单元产生
